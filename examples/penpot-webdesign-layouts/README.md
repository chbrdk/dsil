# Penpot DSIL example (bundled)

Runtime SSOT for the Penpot **example profile** — not DSIL core.

| File | Role |
|------|------|
| `catalog.json` | Components, `layoutRecipe`, instances, bindings |
| `bindings.json` | Penpot layer maps (`source.tool: penpot`) |

Sync to upstream: `npm run sync:dsil-example` → [chbrdk/dsil](https://github.com/chbrdk/dsil) `examples/penpot-webdesign-layouts/`.

Framework spec: [DSIL-SPEC-v1.2](https://github.com/chbrdk/dsil/blob/main/specs/DSIL-SPEC-v1.2.md).
